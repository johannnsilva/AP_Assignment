PahanaEdu - Java Servlet Web Application (starter)

How to use:
1. Edit src/main/java/com/pahanaedu/util/DBUtil.java to set your MySQL username/password and DB URL if needed.
2. Create the database and tables using the SQL file provided (db_init.sql) in MySQL Workbench.
3. Build the WAR with Maven: mvn clean package
4. Deploy the resulting target/pahanaedu.war to Tomcat (or run via your IDE).
5. Default login: admin / admin123 (only if you inserted the user in DB as shown in db_init.sql).
